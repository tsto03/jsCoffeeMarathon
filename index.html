JavaScript Basics Marathon!

A coding marathon is when we, as a whole class will write code together in front of the group one by one. In addition to this, each student will be cloning this code from GitHub and code along as well.

A few things to consider:

  • Everyone is required to do active participation. That means assisting the person who is in front of the group, asking questions and providing feedback. High fives and cheering is encouraged!
  • Making mistakes is a bonus, think of it as learning to debug and fix code.
  • Be sure to rest and stretch so you don't hurt your hands and wrists when you program.
  • If you do not know the answer, you and your team can look it up using any resource you all have.
